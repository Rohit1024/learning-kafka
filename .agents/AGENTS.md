# Project Rules

- **Lesson Location**: When generating or managing lessons for this repository, ALWAYS place them in the `docs/lessons/` directory. Do not place them in a root `lessons/` folder.
- **Lesson Naming**: Follow the existing zero-padded naming convention (e.g., `0001-topic.md`, `0002-another-topic.md`).
- **Lesson Format**: Include "What you will learn" sections, Mermaid diagrams for visual learning, and clear "Gotchas".
