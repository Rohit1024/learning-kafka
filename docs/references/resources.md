# Apache Kafka Mastery Resources

## Knowledge

- [Confluent Developer Platform](https://developer.confluent.io/)
  The definitive modern learning center for Apache Kafka. Use for: learning architecture, pattern guides, and ecosystem definitions.
- [Book: _Kafka: The Definitive Guide (2nd Edition)_ — Gwen Shapira, Todd Palino, Rajini Sivaram, and Krit Petty](https://www.oreilly.com/library/view/kafka-the-definitive/9781492043072/)
  Foundational textbook covering Kafka internal design, broker configuration, and production operations. Use for: deep understanding of broker internals, reliability guarantees, and clustering.
- [Spring for Apache Kafka Documentation](https://spring.io/projects/spring-kafka)
  Official documentation for the Spring-Kafka integration. Use for: `@KafkaListener` configurations, `KafkaTemplate` usage, container factory customisations, and error-handling strategies.
- [Platformatic Kafka Library docs](https://github.com/platformatic/platformatic)
  GitHub repository and documentation for Platformatic packages including Kafka modules. Use for: TypeScript event-driven service setups.

## Wisdom (Communities)

- [Confluent Community Slack](https://slackpass.io/confluentcommunity)
  A highly active Slack workspace with channels dedicated to development, Spring Boot, client libraries, and operations. Use for: real-time troubleshooting, architectural feedback, and networking.
- [r/apachekafka](https://reddit.com/r/apachekafka)
  Subreddit dedicated to Apache Kafka. Use for: industry news, article shares, and community questions.
- [Apache Kafka Mailing Lists](https://kafka.apache.org/contact)
  Official ASF development and user mailing lists. Use for: tracking proposals (KIPs) and deep architectural design discussions.
